# Dashboard

This is the dashboard, assignment for the frontend trainee position.

## Technologies Used

- **React.js**
- **Tailwind CSS**
- **Chart.js**

## Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Mirthhh08/assessment-dashboard
```

### 2. Install dependencies:

```  bash
cd dashboard
npm install
```

### 3. Run the app:

```bash
npm run dev
```
